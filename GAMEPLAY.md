# 🎮 게임 플레이 가이드

## 시작하기

### 1단계: 캐릭터 생성 🎭

게임을 시작하면 캐릭터 생성 화면이 나타납니다.

1. **이름 입력**: 원하는 모험가 이름을 입력하세요
2. **직업 선택**: 
   - **전사** ⚔️: 높은 체력과 물리 공격력
   - **궁수** 🏹: 빠른 공격과 회피
   - **마법사** 🔮: 강력한 마법 공격
3. **모험 시작** 버튼 클릭

---

## 게임 플레이 방법

### 프로액티브 힌트 배너 💡

화면 상단에 파란색 배너가 표시됩니다:
```
┌────────────────────────────────────┐
│ 💡 게시판을 확인하거나 NPC와...    │
└────────────────────────────────────┘
```

이 배너는 현재 할 수 있는 행동을 자동으로 제안합니다.

---

### 방법 1: 선택지 클릭 🖱️

게임이 제공하는 선택지를 클릭하세요:

```
💬 선택하세요:
┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│ 💬 무슨 일이신가요? │  │ ✅ 도와드리겠습니다 │  │ ⏰ 나중에 다시...  │
└──────────────────┘  └──────────────────┘  └──────────────────┘
```

**장점**:
- 간단하고 직관적
- 실수 없이 안전하게 진행
- 초보자 추천

---

### 방법 2: 텍스트 입력 ⌨️ (추천!)

하단 입력창에 원하는 행동을 **자유롭게** 입력하세요:

#### 전투 예시:
```
> 슬라임을 공격한다
> 고블린에게 파이어볼을 쏜다
> 방어 태세를 취한다
```

#### 탐험 예시:
```
> 주변을 조사한다
> 대장간으로 이동한다
> 동쪽 숲을 탐험한다
```

#### 상호작용 예시:
```
> 상인에게 말을 건다
> 체력 포션을 사용한다
> 휴식을 취한다
```

**장점**:
- 완전한 자유도
- 창의적인 플레이 가능
- LLM이 자동으로 이해하고 실행

**팁**: 
- Enter 키로 전송 가능
- 한국어 자연어 입력 지원
- 구체적일수록 좋은 결과

## NPC와 대화하기 💬

### 기본 대화

NPC가 먼저 말을 걸어옵니다:
```
마을 촌장: "오, 새로운 얼굴이군요! 잠깐만 시간을 내주실 수 있겠습니까?"

💬 선택하세요:
[무슨 일이신가요?]  [먼저 쉬고 싶습니다]  [이 마을에 대해 알려주세요]
```

### 신뢰도 시스템 ❤️

NPC와 대화하면 **신뢰도**가 상승합니다:
- 친절한 선택: +5~10
- 도움 제공: +10~20
- 퀘스트 완료: +20~30

**신뢰도 확인**: 
- 0-20: 냉담함
- 20-40: 호의적
- 40-60: 신뢰함
- 60-80: 친밀함
- 80-100: 절친

### NPC 기억 🧠

NPC는 **과거 대화를 기억**합니다:
```
마을 촌장: "지난번에 고블린 소탕을 도와주셨죠? 정말 감사했습니다."
```

**기억되는 것**:
- 대화 내용 (최근 10개)
- 도움을 준 기록
- 선택한 행동
- 약속한 내용

---

## 퀘스트 시스템 📜

### 퀘스트 획득

**조건**: 신뢰도 40 이상

신뢰도가 충분하면 새로운 선택지가 나타납니다:
```
💬 선택하세요:
[무슨 일이신가요?]  [📜 퀘스트를 받습니다]  [나중에 다시 오겠습니다]
```

### 퀘스트 종류

1. **전투 퀘스트** ⚔️
   - "고블린 10마리 처치"
   - "보스 몬스터 토벌"

2. **탐험 퀘스트** 🗺️
   - "던전 최심부 도달"
   - "숨겨진 유적 발견"

3. **수집 퀘스트** 📦
   - "약초 20개 수집"
   - "희귀 광석 채굴"

4. **복합 퀘스트** 🎯
   - "몬스터 처치 + 아이템 수집"
   - "NPC 구출 + 귀환"

### 퀘스트 진행

```
> 슬라임을 공격한다
✅ 슬라임에게 62의 데미지를 입혔습니다!

📜 퀘스트 진행: 슬라임 처치 (1/10)
```

---

## 전투 시스템 ⚔️

### 공격하기

```
> 슬라임을 공격한다
> 고블린에게 스킬을 사용한다
```

**결과**:
```
✅ 슬라임에게 62의 데미지를 입혔습니다!

당신의 검이 슬라임의 젤라틴 같은 몸을 가로지르며 
푸른 액체가 사방으로 튀어올랐습니다.
```

### 방어하기

```
> 방어 태세를 취한다
```

**효과**: 다음 공격 데미지 50% 감소

### 휴식하기

```
> 휴식한다
```

**효과**: HP/MP 완전 회복 (안전한 장소에서만)

## 탐험 및 이동 🗺️

### 위치 이동

```
> 대장간으로 이동한다
> 마을 광장으로 간다
> 동쪽 숲을 탐험한다
```

**사용 가능한 위치**:
- 시작의 마을
- 동쪽 숲
- 어둠의 던전
- 상점
- 대장간

### 주변 조사

```
> 주변을 조사한다
> 상자를 열어본다
```

**발견 가능**:
- 숨겨진 아이템
- 비밀 통로
- 단서
- 보물

---

## 아이템 사용 💊

```
> 체력 포션을 사용한다
> 마나 포션을 마신다
```

**효과**:
- 체력 포션: HP +50
- 마나 포션: MP +30

## 고급 팁 🌟

### 1. 창의적인 입력

**일반적인 입력**:
```
> 공격한다
```

**창의적인 입력**:
```
> 슬라임의 약점인 핵을 노려 공격한다
> 몸을 숨기고 기습 공격을 시도한다
```

LLM이 더 풍부한 내러티브를 생성합니다!

### 2. NPC 관계 관리

- 여러 NPC와 고르게 대화하세요
- 약속은 꼭 지키세요
- 도움을 요청받으면 적극 도와주세요

### 3. 효율적인 퀘스트

- 신뢰도 40 도달 시 즉시 퀘스트 확인
- 비슷한 위치의 퀘스트를 함께 진행
- 퀘스트 보상으로 더 강해지기

### 4. 탐험 전략

- 새로운 지역은 항상 조사하기
- 안전한 루트 파악하기
- 휴식은 안전한 장소에서만

## 자주 묻는 질문 ❓

**Q: 게임이 멈춘 것 같아요**  
A: 60초 무응답 시 상단 힌트가 자동 업데이트됩니다. 힌트를 참고하세요.

**Q: 텍스트 입력이 안 돼요**  
A: Enter 키를 눌러보세요. 전송 버튼(종이비행기 아이콘)도 사용 가능합니다.

**Q: 퀘스트는 어떻게 받나요?**  
A: NPC와 대화하여 신뢰도를 40 이상 올리면 "📜 퀘스트를 받습니다" 버튼이 나타납니다.

**Q: 저장은 어떻게 하나요?**  
A: IndexedDB에 자동 저장됩니다. 브라우저를 닫아도 데이터가 유지됩니다.

**Q: 잘못된 선택을 했어요**  
A: 게임은 계속 진행됩니다. 다음 선택으로 만회할 수 있습니다.
